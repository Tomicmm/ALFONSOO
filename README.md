# ALFONSOO

## 📌 Descripción
Este proyecto consiste en el desarrollo de un sistema/robot, incluyendo su diseño físico, programación y estructura.

## 📂 Estructura del repositorio

- HARDWARE → Componentes físicos, conexiones y electrónica
- SOFTWARE → Código del sistema
- CHASIS 3D → Diseños e impresiones 3D
- LIBRERIAS → Librerías utilizadas en el proyecto
